Syfte

Den här sidan är en skoluppgift. Jag har skapat ett api som hanterar arbetserfarenheter och har byggt en fristående webbplats som visar och skickar data till api:t.

Teknik

  • node.js och express för backend
  • mongodb som databas
  • html css och javascript för frontend
  • parcel som utvecklingsmiljö

Slutsats

Jag har lärt mig hur man skapar ett rest api med express och hur man kopplar det till en mongodb databas. Det var första gången jag arbetade med en nosql databas och det hjälpte mig att förstå skillnaden mot en relationsdatabas. Jag har också övat på att använda fetch api från frontend för att hämta och skicka data. Några utmaningar var kopplingen till Atlas och förståelse för hur databasservern fungerar i molnet, men genom felsökning och tålamod lyckades jag lösa problemen. Det här momentet har gett mig en djupare förståelse för hur en modern webbtjänst fungerar från frontend till backend och databas.
