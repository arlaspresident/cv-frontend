Syfte

Den här sidan är en skoluppgift. Jag har skapat ett api som hanterar arbetserfarenheter och har byggt en fristående webbplats som visar och skickar data till api:t.

Teknik

  • node.js och express för backend
  • mysql som databas
  • html css och javascript för frontend
  • parcel som utvecklingsmiljö

Slutsats

Jag har lärt mig hur man skapar ett rest api med express och hur man kopplar det till en mysql databas. Det var första gången jag byggde en egen webbtjänst från grunden och det hjälpte mig att förstå hur backend fungerar i praktiken. Jag har också övat på att använda fetch api från frontend för att hämta och skicka data. En del saker var lite svåra till en början som till exempel portkrockar och cors men jag löste det med hjälp av felsökning och genom att läsa felmeddelanden noggrant. Jag tycker att det här momentet var väldigt lärorikt eftersom det binder ihop många olika delar som jag har jobbat med tidigare i andra kurser. det känns som att jag börjar få en bättre helhetsbild av hur en webbapplikation fungerar från databas till användargränssnitt.
